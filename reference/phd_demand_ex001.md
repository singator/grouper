# PhD Demand Matrix Example 001

An example demand matrix to use with the PhD workload allocation model.

## Usage

``` r
phd_demand_ex001
```

## Format

### `phd_demand_ex001`

A matrix with 4 rows and 2 columns.

Columns are in the order `TA`, `GR`. Row names store the course codes.
